JAVA MOBİLYA SATIŞ PROJESİ

Bu proje mobilya satış mağazasındaki indirimleri ve ürün adedini hesaplamak için tasarlanmıştır.

Projemin Özellikler:

1- Ürünleri listeler.
2- İndirim uygular.
3- Scanner ile kullanıcı etkileşimi sağlar.

Projemde Kullandıklarım:

1- Java JDK
2- Metotlar = Static metot
3- Karar yaıları = Switch case
